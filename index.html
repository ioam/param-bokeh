ParamBokeh ¶

ParamBokeh is no longer maintained - it has been superseded by Panel

Generate bokeh widgets for parameterized objects

ParamBokeh is an open-source Python library that allows you to easily create GUIs from existing objects (in Jupyter notebooks, Bokeh apps, and potentially more)

The User Guide shows the concepts involved with ParamBokeh and should help you get started using it as quickly as possible.

Please feel free to report issues or contribute code . You are also welcome to chat with the developers on gitter .
